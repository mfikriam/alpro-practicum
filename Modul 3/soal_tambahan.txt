1. Apa fungsi looping pertama pada program bubble sort "for (int i = 5; i >=0; i--)"
Jawaban:
Loop ini mengontrol jumlah iterasi untuk loop luar dari program bubble sort. Loop luar mengakses setiap elemen array mulai dari elemen terakhir dan bergerak menuju elemen pertama. Hal ini dilakukan karena selama iterasi loop luar, algoritma buble sort membandingkan elemen array yang berdekatan dan menukarnya jika urutannya salah. Dengan memulai iterasi dari elemen terakhir, algoritma dapat memastikan bahwa elemen terbesar akan "menggelembung" ke ujung array di setiap loop luar.

4. Jelaskan mengapa huruf kapital lebih kecil daripada huruf kecil
Jawaban:
Hal ini dikarenakan bahasa pemrograman umumnya mengikuti ASCII (American Standard Code for Information Interchange) yang menetapkan nilai numerik untuk setiap karakter. Di ASCII, huruf besar memiliki nilai numerik lebih rendah daripada huruf kecil. Misalnya nilai ASCII untuk huruf besar 'A' adalah 65, sedangkan nilai ASCII untuk huruf kecil 'a' adalah 97.
